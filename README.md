# fluxer-bitwig


## Installing

<details>
<summary><strong>Installing controller for development</strong> (Click to unfold)</summary>

```bash
ln -s ~/Dokumente/Code-Projekte/fluxer-bitwig/target/fluxerbitwig.bwextension ~/Bitwig\ Studio/Extensions
```
Make sure you have `openjdk-17` and `maven` installed to compile the extension.
